# AutoPark
A street parking mobile application built with Kotlin and Jetpack Compose. 

# Project Overview
- Parking machines are widely used within the Melbourne area, but they suffer from inefficiency (H Locke,2020). Aiming to solve these issues, the team is developing a parking application that allows users to pay for their street parking fees across Melbourne quickly and easily.
- The purpose of this app is to enable street parking payment on mobile to increase efficiency, thereby solving the issue of conventional street parking. This will enhance the user street parking experience.

### Technology Used
- Kotlin
- Jetpack Compose
- Retrofit
- Firebase
- MapBox
