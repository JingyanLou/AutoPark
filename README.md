# AutoPark
A street parking mobile application build by Kotlin + Jetpack Compose. 

# Project Overview
- Parking machines are widely used within the Melbourne area, but they suffer from inefficiency (H Locke,2020). Aiming to solve these issues, The team is developing a parking application that allows users to pay for their street parking fees across Melbourne fast and easy to use.
- The purpose of this app is to enable street parking payment on mobile to increase the efficiency, as a result to solve the issue of conventional street parking. Therefore, enhancing the user street parking experience

### Technology Used
- Kotlin
- Jetpack Compose
- Retrofit
- Firebase
- MapBox
